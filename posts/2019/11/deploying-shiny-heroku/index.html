Deploying shiny: Heroku

Here I will be exploring some ways to deploy shiny apps to the web.

I will be deploying a simple shiny app I made quite some time ago, you can find the code in this repo, the app in itself is not important as we are focused only on th edeployment, but if you want to try it locally you are free to do it by checking out the local branch and running cleaning_script.R to setup the data to use in the app.
